# Gastos Bot Multiusuario: amigos solamente

Bot de Telegram para registrar gastos personales con soporte multiusuario.

## Funciones
- Registrar gastos
- Clasificación automática
- Total por usuario
- Análisis por categoría
- Exportar gastos a formato excel 

## Uso

Formato:
Nombre, Prioridad, Monto

Ejemplo:
Comida, Alta, 120

## Stack
- Python
- Telegram Bot API
- PostgreSQL
